HTML

HTML은 웹 브라우저의 여러 내용 중에서 제목과 본문, 이밎, 표와 같은 웹 요소를 알려주는 뼈대와같은 역할을 합니다.

태그

특수문자 코드표

<br>

줄바꿈할때 사용하며 시각적으로만 의미가 있다. <br></br>할필요없이<br>한번만 사용

<p></p>

단락을 표현할때 사용 정보로써 가치가있다. 줄바꿈도 같이되지만 <br>태그에 비해 시각적으로 자유도가 떨어진다.

<img src="위치">

이미지를 넣고싶을때 사용 src같은 부분을 속성이라고 한다.

<ol>
 <li>내용</li>
</ol>

자동으로 li태그 맨 앞에 번호를 매긴다. ol태그를 부모, li태그를 자식 태그라고 한다.

<ul>
 <li>내용</li>
</ul>

자동으로 li태그 맨 앞에 ●를 매긴다. ul태그를 부모, li태그를 자식 태그라고 한다.

<head></head>

본문을 설명하는 태그를 묶는 고위태그

<title></title>

웹페이지의 제목을 설명 head태그 안에 들어간다.

<meta>

문서를 읽는 방식을 지정 ex) <meta charset ="utf-8">

<body></body>

본문들을 묶는 고위태그

<!doctype html>
<html></html>

웹의 최고위 태그로 head와 body태그도 이안에 들어간다.

<a = herf "주소"></a>

보통 사람들이 말하는 링크를 나타낸다.
